# kumujacoby.github.io

Personal project portfolio for Jacoby Young, Engineering and Design Kumu at Kamehameha Schools-Kapālama.

## About

This site showcases educational technology projects that integrate Hawaiian cultural values with modern learning approaches.

## Projects

### [ʻApomanaʻo](https://kumujacoby.github.io/apomanao/)
A suite of Hawaiian language learning tools built with community guidance, featuring culturally-grounded approaches to teaching and learning ʻŌlelo Hawaiʻi.

## Local Development

This is a static site built with HTML, CSS, and JavaScript. To run locally:

1. Clone this repository
2. Open `index.html` in your browser, or
3. Run a local server: `python -m http.server 8000`
