# Kumu Jacoby - Project Portfolio

This repository hosts my educational technology projects and Hawaiian language learning tools.

## Projects

### [ʻApomanaʻo](./apomanao/)
A suite of digital tools designed to support Hawaiian language learning through culturally-grounded approaches. Built with community guidance, these tools embody Hawaiian ways of thinking and learning.

**Tools include:**
- Ka Hakalama - Hawaiian syllabary spelling practice
- Nā Helu Hawaiʻi - Hawaiian number practice
- Heluhelu Puke - Book reading interface
- And more...

[Visit ʻApomanaʻo →](https://kumujacoby.github.io/apomanao/)
